# 📊 Sales Data Analysis

## 📌 Objective
Analyze sales data to understand regional performance and trends.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib

## 📊 Analysis Performed
- Data creation and cleaning
- Grouping sales by region
- Visualization using bar chart

## 📈 Insights
- West region shows higher sales
- South region has consistent performance

## 🚀 Conclusion
This project demonstrates basic data analysis and visualization skills using Python.
